# Season Zero - Best Of Series

Full TMDB metadata & search with top catalogs. Injects an extra season for each series providing the top ranked epsiodes with TMDB ranking data.

https://stremio-tmdb-best-of.onrender.com/configure

Note that due to the nature of how this add on works, some add ons might not be able to play the corrent stream from Season Zero. This does not impact regular series episode metadata.
